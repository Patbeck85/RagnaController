# Comprehensive Controls Guide (v1.2.0)

RagnaController uses a powerful **5-Layer Input System** to map over 40 actions to a single controller.

## 1. The Layer Logic
By using the shoulder buttons and triggers as modifiers, you can access different skill bars in Ragnarok Online.

| Modifier | Layer | RO Skill Bar Equivalent |
| :--- | :--- | :--- |
| **None** | BASE | Basic Actions & Movement |
| **Hold LB / L1** | L1 Layer | F1 - F4 (Primary Skills) |
| **Hold RB / R1** | R1 Layer | F5 - F8 (Secondary Skills) |
| **Hold LT / L2** | L2 Layer | F9 - F12 (Utility) |
| **Hold RT / R2** | R2 Layer | Ctrl + F1 - F4 (Buffs/Renewal Skills) |

## 2. Specialized Pro System
- **L3 + R3 (Click both sticks):** **PANIC HEAL**. Instantly spams the F4 key (Potions) 10 times in 100ms.
- **LT + RT (Hold both triggers):** **RADIAL MENU**. Select consumable items or emotes via the Right Stick and release to use.
- **LB + RB (Hold both bumpers):** **LOOT VACUUM**. Circles the cursor around your character to collect drops.
- **L3 (Single press):** Toggle the specific Combat Engine (Melee/Mage/Support).
- **R3 (Single press):** Snap cursor to target or cast ground spell (Engine dependent).

## 3. Communication
- **Back/Select + R1:** Opens the **Daisy Wheel** (circular on-screen keyboard) for quick in-game chatting without touching your real keyboard.
- **Back/Select + L1:** Toggles Voice Recognition. Speak into your microphone and the text will be automatically typed into the RO chat.
