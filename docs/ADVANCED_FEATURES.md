# Advanced Pro Features Guide (v1.2.0)

RagnaController includes professional automation features for high-end play.

## 1. Radial Quick-Menu (L2 + R2)
**Trigger:** Hold both **Left Trigger (L2)** and **Right Trigger (R2)**.
- **Usage:** Point the **Right Stick** toward the desired item or emote.
- **Execute:** Release the triggers to execute the command.
- **Configuration:** Emotes are built-in natively. You can configure them via the **Radial** button in the main toolbar. No downloads needed!

## 2. Smart Loot Vacuum (L1 + R1)
**Trigger:** Hold both **Left Bumper (L1)** and **Right Bumper (R1)**.
- **Effect:** The cursor performs a high-speed spiral pattern around your character while spamming Left-Click to pick up massive amounts of loot instantly. 

## 3. Panic Emergency Heal (L3 + R3)
**Trigger:** Click both **Thumbsticks** simultaneously.
- **Effect:** Bypasses all standard delays and fires the **F4** key (usually assigned to Potions or Yggdrasil Berries) 10 times within 100ms. Triggers a strong haptic warning to confirm execution.

## 4. Anti-Freeze Input Injection
RagnaController utilizes a multi-threaded, asynchronous input pipeline. Even if Ragnarok Online or anti-cheat software like Gepard Shield temporarily blocks mouse input, the controller engine will **never freeze**. Background logic will safely skip blocked frames and keep your gamepad fully responsive.

## 5. Multi-Platform Support
RagnaController automatically detects if it's running on **Windows** or **Steam Deck (Proton)** and adjusts internal processing accordingly.
